# `Neovim`

To install, clone repo in `$HOME/.config`: `git clone https://github.com/dpi0/nvim $HOME/.config/nvim`

```bash
$ tree -L 2
.
├── init.lua
├── lazy-lock.json
├── LICENSE
├── lua
│   ├── autocmd.lua
│   ├── binds.lua
│   ├── options.lua
│   ├── plugins
│   └── plugins.lua
├── README.md
└── snippets
    ├── all.json
    └── global.json
```
